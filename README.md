# Fin-Copilot

**Fin-Copilot** is a financial co-pilot SaaS product that automates data analysis for SMBs. Upload your spreadsheet and get visual dashboards, insights, and (coming soon) AI-powered predictions.

## Tech Stack
- Frontend: Vite + React + Tailwind CSS
- Backend: Node.js + Express
- Charts: Recharts
- Auth: To be integrated
- AI: GPT-4 integration (upcoming)

## Getting Started

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm run dev
```
